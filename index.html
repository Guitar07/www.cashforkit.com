Enterprise SSDs

Storage Drives

Enterprise SSDs are high-endurance solid state drives built for server and storage array workloads with superior reliability.

  • ✓ Form Factors: 2.5" SAS/SATA, U.2, M.2, E1.S NVMe
  • ✓ Capacities: 480GB, 960GB, 1.92TB, 3.84TB, 7.68TB, 15.36TB
  • ✓ Endurance: Mixed-use and read-intensive DWPD ratings
  • ✓ Brands: Samsung, Intel/Solidigm, Micron, SK Hynix, Kioxia, Western Digital
  • ✓ Use Cases: Server boot drives, database storage, caching, virtualisation, high-IOPS workloads

DDR4 RAM

Server Memory

DDR4 is the fourth generation of DDR memory technology, widely used in enterprise servers and data centres.

  • ✓ Speeds: 2133MHz, 2400MHz, 2666MHz, 2933MHz, 3200MHz
  • ✓ Modules: 16GB, 32GB, 64GB, 128GB, 256GB
  • ✓ Voltage: 1.2V (lower power consumption than DDR3)
  • ✓ Compatibility: Intel Xeon Scalable, AMD EPYC, and older server platforms
  • ✓ Use Cases: Enterprise servers, workstations, data centres, virtualisation

DDR5 RAM

Server Memory

DDR5 is the latest generation of DDR memory, offering significantly improved performance and efficiency for modern enterprise infrastructure.

  • ✓ Speeds: 4800MHz, 5200MHz, 5600MHz, 6000MHz+ (much faster than DDR4)
  • ✓ Modules: 16GB, 32GB, 64GB, 96GB, 128GB, 256GB
  • ✓ Voltage: 1.1V (more energy efficient than DDR4)
  • ✓ Compatibility: Latest Intel Xeon Scalable (Sapphire Rapids+), AMD EPYC Genoa+
  • ✓ Use Cases: High-performance computing, AI/ML workloads, modern data centres, cloud infrastructure
